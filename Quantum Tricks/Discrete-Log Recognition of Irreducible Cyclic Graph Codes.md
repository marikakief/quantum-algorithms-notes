# Discrete-Log Recognition of Irreducible Cyclic Graph Codes

> **Source:** Geraci-Lidar, arXiv:quant-ph/0703023
> **Tags:** #trick #coding-theory #finite-fields #discrete-log #graph-algorithms

## What it does

Recognizes whether a graph's dual cocycle code has the irreducible cyclic structure needed for a Gauss-sum weight algorithm.

## The trick

For a graph $\Gamma=(E,V)$, row-reduce an incidence-derived cycle matroid matrix to

$$
\mathrm{CMM}(\Gamma)=[I_{n-k}\mid X],
$$

where

$$
n=|E|,\qquad k=|E|-|V|+c(\Gamma).
$$

The dual of the cocycle code is generated by

$$
H^T=[-X^T\mid I_k].
$$

Fix a basis of $\mathbb F_{q^k}$ over $\mathbb F_q$ and read the columns of $H^T$ as field elements $g_i\in\mathbb F_{q^k}$. For a primitive element $g$, compute

$$
\log_g(g_i)
$$

for every nonzero column. The code is equivalent to an irreducible cyclic code when these logs form a permuted list of consecutive multiples of

$$
N=\frac{q^k-1}{n},
$$

namely the columns can be put in the form

$$
(1,g^{Nj},g^{2Nj},\ldots,g^{(n-1)Nj})
$$

with $\gcd(n,j)=1$.

Quantum discrete logarithms make this recognition step efficient.

## When to reach for it

Use this when an algebraic property of a code is hidden in a matrix presentation, and the distinguishing feature is geometric progression in a finite-field multiplicative group. Discrete logs turn multiplicative structure into an arithmetic progression test.

## Complexity

The Geraci-Lidar membership test costs

$$
O(|E|\,k^2\log k\log\log k)
$$

quantum time, from computing $|E|$ discrete logs over $\mathbb F_{q^k}$ plus polynomial-time matrix work.

## Caveat

This depends on a quantum discrete-log subroutine, so it is not a classical pre-processing trick. It also recognizes a very specific code family; failing the test does not mean the partition function is classically easy or quantumly inaccessible by other methods.
