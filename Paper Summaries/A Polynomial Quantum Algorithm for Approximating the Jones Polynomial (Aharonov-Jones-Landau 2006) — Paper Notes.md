## The computational problem

**Jones polynomial approximation:** Given a braid $B$ with $n$ strands and $m$ crossings, and an integer $k$, compute an additive approximation of the Jones polynomial $V_{B^{\text{cl}}}(e^{2\pi i/k})$ of the closure of $B$.

Two closure types are considered:
- **Trace closure** $B^{\text{tr}}$: connect top strand $i$ to bottom strand $i$ around the right side.
- **Plat closure** $B^{\text{pl}}$: connect adjacent strand pairs at top and bottom.

The Jones polynomial is #P-hard to evaluate exactly at any non-trivial value of $t$. The question is whether quantum computers can provide useful approximations.

---

## What the paper does

Provides the first explicit, simple quantum algorithm for approximating the Jones polynomial at roots of unity. Prior to this, the existence of such an algorithm was only implicit in the TQFT-based results of Freedman, Kitaev, Larsen, and Wang, which were inaccessible to computer scientists.

The algorithm is polynomial in $n$, $m$, and $k$, and the approximation problem (for plat closure) is BQP-complete. This gives arguably the most natural BQP-complete problem known ŌĆö a combinatorial problem solved not via the Fourier transform but by encoding algebraic structure directly into quantum gates.

The key idea: the Jones polynomial can be computed as a trace on the Temperley-Lieb algebra. Use a *unitary* representation of this algebra (the path model) to map braids to quantum circuits, then estimate the trace via the Hadamard test.

## The mathematical machinery

### The Temperley-Lieb algebra $TL_n(d)$

An algebra generated by $\{1, E_1, \ldots, E_{n-1}\}$ with relations:
1. $E_i E_j = E_j E_i$ for $|i-j| \geq 2$
2. $E_i E_{i\pm 1} E_i = E_i$
3. $E_i^2 = d \cdot E_i$

The braid group $B_n$ maps into $TL_n(d)$ via $\sigma_i \mapsto AE_i + A^{-1}I$ where $d = -A^2 - A^{-2}$.

### The Jones polynomial as a trace

The Markov trace $\text{tr}: TL_n(d) \to \mathbb{C}$ is the unique linear function satisfying:
1. $\text{tr}(1) = 1$
2. $\text{tr}(XY) = \text{tr}(YX)$
3. $\text{tr}(XE_{n-1}) = \frac{1}{d}\text{tr}(X)$ for $X \in TL_{n-1}(d)$ (the Markov property)

The Jones polynomial is then:

$$
V_{B^{\text{tr}}}(e^{2\pi i/k}) = (-A)^{3w(B^{\text{tr}})} d^{n-1} \text{tr}(\rho_A(B))
$$

where $w$ is the writhe and $A = e^{-\pi i/(2k)}$.

### The path model representation

For parameter $k$, let $G_k$ be a line graph with $k-1$ vertices. Paths of length $n$ on $G_k$ starting from the leftmost vertex form an orthonormal basis for the representation space $V_{n,k}$.

A Temperley-Lieb generator $E_i$ maps to a matrix whose entries are determined by local "cap-cup" pictures: each local maximum/minimum of a Kauffman diagram is assigned a weight depending on the labels of surrounding regions. The resulting representation $\tau(TL_n(d))$ acts on the path space.

When $A = e^{-\pi i/(2k)}$ (so $|A| = 1$), the generators $E_i$ are Hermitian in this representation, making the braid group representation **unitary**. This is what enables the quantum algorithm.

## The algorithms

### Plat closure algorithm

1. Prepare the state $|\alpha\rangle = |1, 0, 1, 0, \ldots, 1, 0\rangle$ (encoding $n/2$ cap-cup pairs as paths).
2. Apply the unitary $Q(B) = \varphi(B)$ ŌĆö the path-model image of the braid ŌĆö using the gate decomposition of each crossing into 2-qubit gates.
3. Estimate $\text{Re}\langle\alpha|Q(B)|\alpha\rangle$ and $\text{Im}\langle\alpha|Q(B)|\alpha\rangle$ via the [[Hadamard Test for Trace Estimation|Hadamard test]].
4. Average over $\text{poly}(n,m,k)$ repetitions, scale by $(-A)^{3w(B^{\text{pl}})} d^{3n/2-1} \lambda_1 / N$.

### Trace closure algorithm

1. Classically sample a random path $p \in P_{n,k}$ with probability $\Pr(p) \propto \lambda_\ell$ (where $\ell$ is the endpoint).
2. Estimate $\langle p|Q(B)|p\rangle$ via the [[Hadamard Test for Trace Estimation|Hadamard test]].
3. Average over repetitions, scale by $(-A)^{3w(B^{\text{tr}})} d^{n-1}$.

Both algorithms run in time $\text{poly}(n, m, k)$. Each crossing contributes one 2-qubit gate (acting on adjacent path qubits), so the total circuit depth is $m$.

## Key results

**Theorem 1.1 (Trace closure):** The algorithm outputs $r$ with $|r - V_{B^{\text{tr}}}(e^{2\pi i/k})| < \varepsilon d^{n-1}$ where $\varepsilon = 1/\text{poly}(n,k,m)$.

**Theorem 1.2 (Plat closure):** The algorithm outputs $r$ with $|r - V_{B^{\text{pl}}}(e^{2\pi i/k})| < \varepsilon d^{3n/2}/N$ with $\varepsilon = 1/\text{poly}(n,k,m)$.

**Theorem 1.3 (BQP-hardness, from Freedman-Larsen-Wang):** The plat closure approximation problem is BQP-hard for constant $k = 5$ and $k \geq 7$. (Later extended to polynomial $k$ by [[The BQP-Hardness of Approximating the Jones Polynomial (Aharonov-Arad 2011) ŌĆö Paper Notes|Aharonov-Arad]].)

The approximation is **additive** ŌĆö the error window has size $\sim d^{n-1}$, independent of the actual value being approximated. This is weaker than multiplicative approximation, but the BQP-hardness result shows it captures the full power of quantum computation nonetheless.

## Limits / caveats

- The approximation is additive, not multiplicative. The exact value of $V_L(t)$ can be exponentially smaller than the approximation window $d^{n-1}$, making the result trivial for some links. BQP-hardness guarantees this isn't always the case, but doesn't tell you *which* links are non-trivial.
- For the trace closure, BQP-hardness is *not known* ŌĆö only for the plat closure. The transformation from plat to trace closure can increase the strand count, degrading the approximation exponentially.
- The algorithm requires $k \geq 5$ (and $k \neq 6$ for BQP-hardness). At $k \leq 4$ or $k = 6$, the representation isn't rich enough for universality.
- The path model representation dimension grows exponentially with $n$ but the algorithm only works in the path subspace ŌĆö no explicit exponential resources needed.

## Reusable ideas

1. **[[Hadamard Test for Trace Estimation]]:** To estimate $\langle\alpha|U|\alpha\rangle$ for a unitary $U$: prepare $\frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)|\alpha\rangle$, apply controlled-$U$, Hadamard the control, measure. The expectation is $\text{Re}\langle\alpha|U|\alpha\rangle$. Repeat with $\frac{1}{\sqrt{2}}(|0\rangle - i|1\rangle)$ for the imaginary part. Simple and widely used ŌĆö this is the standard tool for estimating matrix elements of unitaries.

2. **[[Algebraic Gate Design via Temperley-Lieb Representations]]:** Instead of encoding the problem into a quantum circuit via generic techniques (like Hamiltonian simulation or LCU), encode the algebraic structure directly into the gates. Each braid generator maps to a 2-qubit gate via the TL representation. The gate *is* the problem. This produces extremely shallow circuits ($m$ depth for $m$ crossings) and avoids all overhead from generic simulation.

3. **[[Markov Trace Uniqueness for Jones Polynomial Computation]]:** The Markov trace on $TL_n(d)$ is uniquely determined by three properties (normalisation, cyclicity, and the Markov property). So to compute the Jones polynomial, you don't need the original TL representation ŌĆö *any* representation that supports a trace with these properties will give the same answer. This is the conceptual core that makes the algorithm work: the path model representation is convenient for quantum computing, and uniqueness guarantees it computes the right thing.
