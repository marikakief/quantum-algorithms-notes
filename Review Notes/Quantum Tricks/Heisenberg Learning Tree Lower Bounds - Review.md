# Review: Heisenberg Learning Tree Lower Bounds

Source note: [[Heisenberg Learning Tree Lower Bounds]]

## Primary Sources Checked

- Kannan, Putterman, and Cotler, arXiv:2605.02057, checked current arXiv v1.

## Verdict

Minor issues. The proof-method card is useful, but the Heisenberg-picture notation should be tightened.

## Line-Anchored Comments

- Lines 12-32: the adaptive-tree explanation is good.
- Lines 18-22: if this is called a Heisenberg-picture method, be explicit about whether `Phi_u` is acting on states or whether the dual channel is acting on measurement/distinguishability operators. As written it looks more Schrödinger than Heisenberg.
- Lines 34-44: the permutation-operator example is a good anchor.
- Lines 57-61: the caveat is appropriate. This is a lower-bound method, not an upload protocol.

## Missing or Suggested Additions

- Add one line showing the adjoint-channel form used in the actual proof. That would resolve the notation ambiguity.
